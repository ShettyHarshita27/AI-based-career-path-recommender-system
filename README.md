# AI-based-career-path-recommender-system

An AI-powered web app that helps students choose the right career path by analyzing their skills and interests. Developed with React 18 and Firebase, it provides personalized recommendations through an interactive assessment and secure data handling. 



## Problem Statement

Students often struggle to choose the right career due to lack of guidance and awareness.



## Solution

This system analyzes user input and provides personalized career recommendations using AI logic.



## Key Features 

- User Authentication (Login/Register)
- Career Assessment Form
- AI-based Recommendation Logic
- Firebase Firestore Database
- Dashboard with Results



## Tech Stack

Frontend:
- React 18
- JavaScript

Backend:
- Firebase Authentication
- Firestore Database



## How It Works

1. User registers/login
2. Fills assessment form
3. Data stored in Firebase
4. AI processes inputs
5. Career recommendation generated
